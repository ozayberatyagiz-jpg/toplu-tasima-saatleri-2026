# RouteX Otobüs Bilet Sistemi

## Proje Hakkında
Bu proje Python FastAPI kullanılarak geliştirilmiş modern bir otobüs bileti rezervasyon sistemidir.

Kullanıcılar:
•⁠  ⁠Sefer arayabilir
•⁠  ⁠Koltuk seçebilir
•⁠  ⁠Bilet oluşturabilir
•⁠  ⁠PNR kodu alabilir
•⁠  ⁠Biletlerini görüntüleyebilir

Admin panelinde:
•⁠  ⁠Toplam bilet sayısı
•⁠  ⁠Toplam gelir
•⁠  ⁠Sefer bilgileri görüntülenebilir
