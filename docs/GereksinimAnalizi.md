# Gereksinim Analizi Dokümanı

## Proje Adı
Toplu Taşıma Saatleri Uygulaması

## Proje Amacı
Bu projenin amacı, kullanıcıların toplu taşıma sefer saatlerini kolayca ekleyebilmesi ve görüntüleyebilmesidir. Kullanıcı web arayüzü üzerinden şehir/güzergah ve saat bilgisi girerek yeni sefer oluşturabilir.

## Hedef Kullanıcılar
- Öğrenciler
- Günlük toplu taşıma kullanan kişiler
- Sefer saatlerini takip etmek isteyen kullanıcılar

## Fonksiyonel Gereksinimler
1. Kullanıcı şehir veya güzergah bilgisi girebilmelidir.
2. Kullanıcı sefer saati seçebilmelidir.
3. Kullanıcı “Sefer Ekle” butonu ile yeni sefer ekleyebilmelidir.
4. Eklenen seferler ekranda listelenmelidir.
5. Boş alan bırakıldığında kullanıcı uyarılmalıdır.

## Fonksiyonel Olmayan Gereksinimler
1. Uygulama basit ve anlaşılır bir arayüze sahip olmalıdır.
2. Uygulama tarayıcı üzerinden çalışmalıdır.
3. Kodlar okunabilir ve düzenli olmalıdır.
4. Projede nesne yönelimli programlama mantığı kullanılmalıdır.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## OOP Kullanımı
Projede JavaScript class yapısı kullanılmıştır.

Kullanılan sınıflar:
- Sefer
- OtobusSeferi
- TopluTasimaSistemi

OtobusSeferi sınıfı, Sefer sınıfından kalıtım almaktadır.
